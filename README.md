# SubX - Financial Terminal & Portfolio Tracker

SubX is a Python-based desktop application built with Tkinter, SQLite, and yfinance. It provides a unified dashboard to manage monthly subscriptions and monitor real-time financial markets, including global stocks, cryptocurrencies, and FX pairs.

## Key Features
*   **Subscription Management:** Track recurring expenses and automatically calculate their local currency equivalents using live exchange rates.
*   **Portfolio Tracking:** Monitor real-time prices for US stocks, the Turkish stock market (BIST), and cryptocurrencies via the `yfinance` API.
*   **Interactive Charts:** Visualize live market data and historical price trends using `matplotlib`.
*   **Local Database:** Securely store user credentials, portfolio holdings, and transaction history using `sqlite3`.
